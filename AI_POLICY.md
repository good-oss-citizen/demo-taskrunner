# AI Policy for taskrunner

## General Policy

We accept AI-assisted contributions with the following requirements:
- Disclose AI tool usage in your PR description (which tool, what it helped with)
- You must understand every line of code you submit
- You must be able to respond to review feedback without relying on AI to explain your own code

## Good First Issues

**AI tools must NOT be used to solve issues labeled `good first issue`.**

These issues exist as learning opportunities for new contributors. Using AI to solve
them defeats their purpose — the contributor learns nothing, and we lose an onramp
for growing our community.

If you are a new contributor who wants to use AI tools, we suggest:
- First, solve a `good first issue` on your own to learn the codebase
- Then, use AI tools on more complex issues where the AI assists rather than replaces
  your understanding

Contributors found using AI on good-first-issue PRs will have their PR closed.

## Disclosure Format

In your PR description, include a section:
```
### AI Assistance
- Tool: [name and version]
- Used for: [what the AI helped with]
- Human-written: [what you wrote yourself]
```
